🚀 How to Create EXE

Quick Start (Windows)

  1. Download the package above and extract it
  2. Install Node.js from nodejs.org
  3. Double-click build-windows.bat in the extracted folder
  4. Wait for build to complete (5-10 minutes)
  5. Find your EXE in dist/packages/ folder

📦 What You'll Get After Building

  • Multi-Account Ledger Setup.exe - Full installer with shortcuts
  • Multi-Account Ledger.exe - Portable version (no install needed)
  • win-unpacked/ - Development build folder
